Search Search has clear button like text control. Also has special onscreen keyboard that show magnifying glass for instead of Enter symbol, but not in Chrome.
Tel Tel has clear button like text control. Also has special onscreen keyboard to facilitate phone number entry, but not in Chrome.
URL URL has clear button like text control. Also has special onscreen keyboard to facilitate URL entry, but not in Chrome.
Email Email has clear button when it contains something (other than the placeholder text) and it controls which on screen keyboard layout is used.
Password Password has reveal button when it contains something (other than the placeholder text) completely input by the user and it hasn't lost focus
Date Date has Fluent date picker popup in Edge and a traditional calendar looking control in Chrome.
Time Time has no popup in Chrome but has spinning select elements in a popup for touchability in Edge.
Datetime-local Combines date and time popups in Edge; in Chrome there is no time selection in popup.

input type=button same
input type=reset same
input type=submit same
input type=image it's an image
input type=file Appearance differs, both show classic file open dialog (which is touch friendlier in tablet mode).
input type=color In-page appearance echos buttons. Popups: Edge custom touch friendly color picker with light dismiss; Chrome has modal win32 color picker
input type=radio Appearance differs. Edge: flat, Chrome: shaded. Parts can be styled in Edge with ::-ms- pseudo elements.
input type=checkbox Appearance differs. Edge: flat, Chrome: shaded. Parts can be styled in Edge with ::-ms- pseudo elements. Edge treats double-click as single-click.
input type=range Appearance differs. Orientation change differences. Edge: flat, Chrome: shaded. Parts can be styled in Edge with ::-ms- pseudo elements.
